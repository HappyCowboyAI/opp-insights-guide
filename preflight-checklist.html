Fixes Quick Fixes Common issues and solutions
⚠
Table shows 0 rows
Filters may be too restrictive for this customer's data. Most common: the engagement_level filter. Try removing the engagement filter first to see if data comes back, then add filters incrementally.
⚠
Engagement scores are all blank
OOTB engagement scores may not be enabled for this customer. Check with Backstory support — engagement scoring requires activity capture to be running for at least 7 days.
⚠
Bookmarklet doesn't work
URL too long for this browser. Use the console script instead (recommended for all tables). Open DevTools, paste the script, and press Enter.
⚠
Error: Not logged in
You must be on app.people.ai and logged in before running the script. Navigate there first, verify you see the dashboard, then run the script.
