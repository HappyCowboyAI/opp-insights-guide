How to Use This Guide

Three options per artifact — pick whichever works best for you.

Option A — Bookmarklet
Drag & Click
Drag the purple button to your bookmarks bar. Navigate to app.people.ai, click the bookmark. Table/board creates and opens automatically.
Option B — Console Script
Copy & Paste
Expand the console script, copy it, open DevTools on app.people.ai (⌘+Option+J), paste into the Console tab and press Enter. New to the browser console? See our step-by-step guide →
After Each Deploy
Record the ID
Each script logs the created table ID. Paste it into the ID field below the card — this feeds the board generator in Step 3.
